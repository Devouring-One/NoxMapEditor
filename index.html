    • Nox Map Editor

    The unofficial map editor for Westwood Online's Nox game

    Project maintained by NoxNet Hosted on GitHub Pages — Theme by mattgraham

    Requirements

    • Visual Studio 2012 Professional or higher

    Building

    1. Clone or download the repo
    2. Open the solution file (.sln) with Visual Studio
    3. Click Build -> Solution

    If the build fails, please create an issue on the main Github page. This will ensure that we are notified of the bug and can work on a fix.

    Support

    The main line of help and support is available on the Nox forums. There you will find the English speaking Nox community.
